Z3: An Efficient SMT Solver

Z3 is a new high-performance theorem prover being developed at
Microsoft Research. Z3 supports linear real and integer arithmetic,
fixed-size bit-vectors, extensional arrays, uninterpreted functions,
and quantifiers. 

Documentation: doc\index.html

Binaries:
  x86 binaries: bin\
  x64 binaries: x64\


For more information, visit the Z3 website:
http://research.microsoft.com/projects/z3
