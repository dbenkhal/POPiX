00219 // Pseudo-predicates are "predicates" defined using the DEFPRED command.
00220 // They are not really predicates, but functions. So, the following commands
00221 // are valid in Simplify:
00222 //
00223 // (DEFPRED (P x))
00224 // (BG_PUSH (EQ (f (P x)) x))
00225 // (BG_PUSH (EQ (P x) (P y)))
00226 // (BG_PUSH (AND (P 0) (P 1)))
00227 //
00228 // In other words, a pseudo-predicate can be used in contexts where a term or fomula
00229 // is expected. When a formula is expected, Simplify adds an equality to |@true|.
00230 // For example, the last command above is mapped to:
00231 //
00232 // (BG_PUSH (AND (EQ (P 0) |@true|) (EQ (P 1) |@true|)))
00233 //
00234 
00235 enum simplify_decl_kind {
00236     SD_FUNCTION,
00237     SD_PREDICATE,
00238     SD_PSEUDO_PREDICATE
00239 };
