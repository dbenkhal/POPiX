Download

iZ3 is available for x86 Windows and Linux, and can be downloaded as part of the Microsoft Research Z3 distribution.

Installation for Windows

iZ3 installs, along with Z3, into your Program Files folder, in a subfolder "Microsoft Research/Z3.X.Y", where X.Y is the version number. Put this folder in your PATH.

iZ3 requires a component "libfoci.dll", which can be obtained here. This library should be installed either in "Microsoft Research/iZ3.X.Y" or somewhere in your path.

Installation for Linux

Expand the archive in a suitable location, and make sure that the "bin" subdirectory is in your PATH and the "lib" subdirectory is in your LD_LIBRARY_PATH.

iZ3 requires a component "libfoci.so", which can be obtained here. This library should be installed somewhere in your LD_LIBRARY_PATH.
